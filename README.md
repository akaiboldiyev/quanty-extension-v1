Quanty
This is a productivity extension.

How to install:

1. Download zip from GitHub and unzip.
2. Chrome → chrome://extensions → Enable "Developer mode" top right.
3. Click "Load unpacked" → select the extension folder.
4. Pin Quanty icon. Done.

Requires Chrome 114+ / Edge. Win 10/11 or Mac.

Quanty
Это расширение для продуктивности.

Как установить:

1. Скачай zip с GitHub и распакуй.
2. Chrome → chrome://extensions → Включи "Режим разработчика" справа сверху.
3. Жми "Загрузить распакованное" → выбери папку с расширением.
4. Закрепи иконку Quanty. Готово.

Нужен Chrome 114+ / Edge. Win 10/11 или Mac..
